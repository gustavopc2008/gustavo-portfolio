═══════════════════════════════════════════════════════════════
  EXEMPLO DE CONFIGURAÇÃO DNS VERCEL
═══════════════════════════════════════════════════════════════

Quando você adiciona um domínio na Vercel, ela mostra algo assim:

┌─────────────────────────────────────────────────────────────┐
│ CONFIGURAÇÃO PARA: gustavodesign.tech                       │
└─────────────────────────────────────────────────────────────┘

Adicione os seguintes registros DNS:

┌──────┬──────┬──────────────────────┬────────┐
│ Tipo │ Nome │ Valor                │ TTL    │
├──────┼──────┼──────────────────────┼────────┤
│ A    │ @    │ 76.76.21.21          │ Auto   │
│ A    │ @    │ 76.223.126.88        │ Auto   │
│ A    │ @    │ 13.248.148.5          │ Auto   │
│ A    │ @    │ 13.248.148.4          │ Auto   │
└──────┴──────┴──────────────────────┴────────┘

OU (dependendo da Vercel):

┌────────┬──────┬──────────────────────┬────────┐
│ Tipo   │ Nome │ Valor                │ TTL    │
├────────┼──────┼──────────────────────┼────────┤
│ A      │ @    │ 76.76.21.21          │ Auto   │
│ CNAME  │ www  │ cname.vercel-dns.com │ Auto   │
└────────┴──────┴──────────────────────┴────────┘

┌─────────────────────────────────────────────────────────────┐
│ O QUE FAZER NA HOSTINGER:                                  │
└─────────────────────────────────────────────────────────────┘

1. Vá em: Domínios → gustavodesign.tech → DNS

2. DELETE registros antigos (se houver):
   - Qualquer A que aponte para IP da Hostinger
   - Qualquer CNAME de www

3. ADICIONE os registros que a Vercel mostrar

4. SALVE cada registro

5. AGUARDE 5-30 minutos

6. No Vercel, clique em "Refresh"

═══════════════════════════════════════════════════════════════
  IMPORTANTE:
═══════════════════════════════════════════════════════════════

⚠️ Use os valores EXATOS que a Vercel mostrar!
⚠️ Não invente valores - copie da Vercel!
⚠️ DNS pode demorar até 24h (mas geralmente 5-30min)
