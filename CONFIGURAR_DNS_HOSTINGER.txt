  CONFIGURAR DNS NA HOSTINGER PARA VERCEL
═══════════════════════════════════════════════════════════════

❌ PROBLEMA:
   Domínios mostrando "Invalid Configuration" na Hostinger
   Isso é normal! Precisamos configurar o DNS.

✅ SOLUÇÃO:
   Configurar DNS na Hostinger para apontar para Vercel

┌─────────────────────────────────────────────────────────────┐
│ PASSO 1: PEGAR INFORMAÇÕES DA VERCEL                       │
└─────────────────────────────────────────────────────────────┘

1. No Vercel, vá em Settings → Domains

2. Você deve ter adicionado:
   - gustavodesign.tech
   - www.gustavodesign.tech

3. A Vercel vai mostrar instruções de DNS tipo:
   
   Tipo: A
   Nome: @
   Valor: 76.76.21.21
   
   Tipo: CNAME
   Nome: www
   Valor: cname.vercel-dns.com

   (Os valores podem ser diferentes - use os que a Vercel mostrar!)

┌─────────────────────────────────────────────────────────────┐
│ PASSO 2: CONFIGURAR DNS NA HOSTINGER                        │
└─────────────────────────────────────────────────────────────┘

1. No hPanel da Hostinger, vá em:
   → Domínios → Gerenciar → DNS

2. OU vá em:
   → Domínios → gustavodesign.tech → DNS

3. Você vai ver uma lista de registros DNS

4. PROCURE e DELETE (se existir):
   - Qualquer registro A que aponte para IP da Hostinger
   - Qualquer registro CNAME de www

5. ADICIONE os registros que a Vercel pediu:

   REGISTRO 1 (para gustavodesign.tech):
   Tipo: A
   Nome: @ (ou deixe vazio, ou coloque o domínio)
   Valor: IP que a Vercel deu (ex: 76.76.21.21)
   TTL: 3600 (ou padrão)
   [Salvar]

   REGISTRO 2 (para www.gustavodesign.tech):
   Tipo: CNAME
   Nome: www
   Valor: cname.vercel-dns.com (ou o que a Vercel pedir)
   TTL: 3600 (ou padrão)
   [Salvar]

┌─────────────────────────────────────────────────────────────┐
│ PASSO 3: AGUARDAR PROPAGAÇÃO                               │
└─────────────────────────────────────────────────────────────┘

⚠️ IMPORTANTE: DNS pode demorar até 24 horas para propagar!

Normalmente funciona em:
- 5-30 minutos (mais comum)
- Até 2 horas
- Raramente até 24 horas

┌─────────────────────────────────────────────────────────────┐
│ PASSO 4: VERIFICAR NA VERCEL                                │
└─────────────────────────────────────────────────────────────┘

1. No Vercel, vá em Settings → Domains

2. Clique em "Refresh" nos domínios

3. Aguarde alguns minutos

4. O status deve mudar de "Invalid Configuration" para:
   ✅ "Valid Configuration" ou "Configured"

┌─────────────────────────────────────────────────────────────┐
│ EXEMPLO DE CONFIGURAÇÃO DNS:                                │
└─────────────────────────────────────────────────────────────┘

Na Hostinger, você deve ter algo assim:

┌──────────┬──────┬─────────────────────┬────────┐
│ Tipo     │ Nome │ Valor               │ TTL    │
├──────────┼──────┼─────────────────────┼────────┤
│ A        │ @    │ 76.76.21.21         │ 3600   │
│ CNAME    │ www  │ cname.vercel-dns.com│ 3600   │
└──────────┴──────┴─────────────────────┴────────┘

⚠️ Use os valores EXATOS que a Vercel mostrar!

═══════════════════════════════════════════════════════════════
  PROBLEMAS COMUNS:
═══════════════════════════════════════════════════════════════

❌ Ainda mostra "Invalid Configuration"
   → Aguarde mais tempo (DNS demora)
   → Verifique se os valores estão corretos
   → Clique em "Refresh" na Vercel

❌ Site não carrega
   → Aguarde propagação do DNS
   → Verifique se os registros estão salvos
   → Limpe cache do navegador

❌ Não encontro onde configurar DNS
   → hPanel → Domínios → Seu domínio → DNS
   → Ou procure por "DNS" no menu
